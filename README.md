# Kestros Build Helper

Manages the build order for Kestros

### Build Master
```
git clone --recurse-submodules git@github.com:kestros/kestros-build-helper.git &&
mvn clean install
```

### Build Develop
```
git checkout develop &&
git clone --recurse-submodules git@github.com:kestros/kestros-build-helper.git &&
mvn clean install
```
